# shield-doors-windows

A WordPress website for Shield Doors & Windows, showcasing premium doors, windows, and related services.

## Features

- Responsive design
- Product showcase
- Service pages
- Contact form
- SEO-friendly structure
- WordPress CMS

## Technologies Used

- WordPress
- PHP
- HTML5
- CSS3
- JavaScript
- MySQL

## Requirements

- PHP 8.1+
- MySQL 5.7+ or MariaDB
- Apache or Nginx
- WordPress 6.x

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/shield-doors-windows.git
   ```

2. Copy the project to your web server.

3. Create a MySQL database.

4. Import the database.

5. Configure the `wp-config.php` file.

6. Open the website in your browser.

## Folder Structure

```
├── themes/
├── plugins/
└── uploads/
```
